-- Работа с дампами в Windows
https://www.youtube.com/watch?v=QDvf1y-gl6Q
Дампом - файл с полным или частичным содержимым памяти компьютера или базы данных в момент создания этого файла.
Дамп памяти делается, как правило, для отладки программ, поскольку позволяет выяснить содержимое памяти в любой 
заданный момент времени.
Дамп таблиц базы данных нужен либо для создания бэкапа (резервной копии) таблиц, на случай взлома или потери сервера, 
либо для переноса базы данных на другой сервер. Cлово «демпинг» происходит от того же самого английского глагола.
